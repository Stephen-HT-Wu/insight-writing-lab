# Argumentation guide

Build the article around a real tension. Define ambiguous terms before relying on them. For each major claim, identify evidence, warrant, scope, strongest objection, and resulting level of confidence.

Avoid common failures:

- Treating correlation as causation.
- Generalizing beyond a study's population or period.
- Selecting only evidence that supports the initial intuition.
- Replacing an opponent's strongest view with an easy caricature.
- Presenting a false balance when evidence is asymmetric.
- Using abstraction to hide a missing mechanism.

An insightful conclusion should state what changes once the evidence is considered: a distinction becomes necessary, a trade-off becomes visible, a common framing fails, or a new question becomes unavoidable.
